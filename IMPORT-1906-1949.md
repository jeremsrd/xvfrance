# IMPORT — Matches 1906-1949 depuis equipe-france.fr

## Contexte

La base contient actuellement ~925 matches (1950-2025). Il manque la période
1906-1949 (~100 tests). Le site equipe-france.fr a toutes les données
année par année, accessibles via l'URL :

https://www.equipe-france.fr/rugby/masculin/{année}

Ex: https://www.equipe-france.fr/rugby/masculin/1906

## Objectif

Créer une commande Artisan qui scrape equipe-france.fr pour les années 1906 à 1949
et importe les matches manquants dans la base.

## Étape 1 — Analyser la structure HTML

Commence par fetcher la page https://www.equipe-france.fr/rugby/masculin/1949
(la plus récente de la période manquante) et analyse le HTML pour identifier :
- Comment les matches sont listés (structure DOM, classes CSS)
- Où se trouvent : date, adversaire, score France, score adversaire, compétition,
  lieu (domicile/extérieur)

Affiche la structure identifiée avant de continuer.

## Étape 2 — Créer la commande de scraping

```bash
php artisan make:command ImportHistoricalMatches
```

La commande doit :

### a) Boucler sur les années 1906 à 1949
- Sauter les années sans matches internationaux (1915-1919 : Première Guerre mondiale,
  1940-1945 : Seconde Guerre mondiale)
- Années à scraper : 1906-1914, 1920-1939, 1945-1949
- Note : 1945 et 1946 peuvent ne pas avoir de matches officiels

### b) Pour chaque année, fetcher la page et parser les matches
- Utiliser le package HTTP de Laravel (Http::get) pour fetcher
- Parser le HTML avec un parser DOM (installer symfony/dom-crawler si besoin :
  `composer require symfony/dom-crawler`)
- Extraire : date, adversaire, score France, score adverse, compétition

### c) Pour chaque match, déterminer :
- `opponent` : chercher ou créer dans la table countries
  (mapping noms français → anglais si besoin)
- `france_score` / `opponent_score`
- `is_home` : déduire si France joue à domicile (match en France) ou à l'extérieur
- `competition` : mapper vers les compétitions existantes
  - "Tournoi des Cinq Nations" → competition "Tournoi des 5/6 Nations"
  - "Test match" ou "Match amical" → NULL ou TEST
  - "Jeux Olympiques" → créer une nouvelle compétition si nécessaire
- `match_date` : parser la date

### d) Vérifier les doublons
- Ne pas importer si un match existe déjà à la même date contre le même adversaire

### e) Afficher un résumé par année
- Année | Matches trouvés | Matches importés | Doublons ignorés

## Étape 3 — Options de la commande

```bash
# Import complet 1906-1949
php artisan import:historical

# Import d'une année spécifique (pour test)
php artisan import:historical --year=1906

# Dry run
php artisan import:historical --dry-run

# Avec délai entre les requêtes (respecter le serveur)
php artisan import:historical --delay=2
```

## Notes importantes

- **Respecter le serveur** : ajouter un délai de 1-2 secondes entre chaque requête
  (sleep(2) entre chaque année)
- **User-Agent** : utiliser un User-Agent descriptif :
  "xvfrance.fr Historical Import Bot / 1.0"
- Les matches de cette période sont principalement :
  - Tournoi des Cinq Nations (depuis 1910)
  - Tests contre les nations du Sud (NZ, Australie, Afrique du Sud)
  - Quelques matches amicaux / tournées
  - Matches olympiques (1920, 1924)
- Les adversaires principaux : Angleterre, Écosse, Pays de Galles, Irlande,
  Nouvelle-Zélande, Afrique du Sud, Australie, Allemagne, Roumanie, USA
- La France a été exclue du Tournoi de 1932 à 1939 (accusation de professionnalisme)
  mais a continué à jouer des matches amicaux
- Pas de matches internationaux de 1940 à 1944 (Seconde Guerre mondiale)
- Le Tournoi a repris en 1947
